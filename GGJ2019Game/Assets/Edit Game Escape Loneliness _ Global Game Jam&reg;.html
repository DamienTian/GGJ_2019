Edit Game Escape Loneliness

Game Information
This game is about finding similar characters, form a new family and survive from the "Loneliness".

Inside the game world, the player needs to explore this dark "Loneliness world" and find shapes that are same as player's shape. Finding more shape can help player him/herself to survive in the world and help others to survive in this "Lonely world".

The theme of our game is abstractly related to the THEME of this year: we try to represent the "home" with a group that shares similarities, the group members support each other and go through hard times. The win condition: In order to win this game, the player needs to find enough "homie" (similar shape) and survive in the game world. The lose conditions: Player's loneliness reaches the 100 before enough "homie" were found.
